การสำรวจที่ไร้ประโยชน์ แต่น่าสนใจ

เว็บไซต์อย่างเป็นทางการของการค้นพบที่ (อาจ) ไม่มีจุดประสงค์

🧃 ทำไมกล่องน้ำผลไม้ถึงไม่มีมุมที่นุ่มนวล?

การศึกษาตามหลักสรีรศาสตร์กำลังดำเนินการอยู่ หลังจากชนเข้ากับกล่องที่ตกลงมาจากตู้เย็น ผู้เขียนได้ริเริ่มโปรโตคอลการทดสอบตามความเจ็บปวดที่เกิดจากบรรจุภัณฑ์ที่แตกต่างกัน ผลเบื้องต้น: มุมซ้ายล่างอันตรายที่สุด

🪑 การวิจัยเมื่อวันที่ 12 มีนาคม 2023 — เก้าอี้ vs สตูล

การทดลอง: ทำงาน 8 ชั่วโมงโดยนั่งบนเก้าอี้สตูล แล้วเปลี่ยนเป็นเก้าอี้

ผลลัพธ์: หลังชนะ แต่เกียรติยศต้องทนทุกข์ทรมาน
สรุป: สตูลเป็นการลงโทษตัวเองในรูปแบบของเฟอร์นิเจอร์

🦕 ความเร็วของไดโนเสาร์บนสกู๊ตเตอร์ (สมมติฐาน)

หลังจากการจำลองบนกระดาษหลายครั้ง เราคาดว่าเวโลซีแรปเตอร์ที่ติดตั้งสกู๊ตเตอร์ไฟฟ้าอาจมีความเร็วถึง 40 กม./ชม. หากรู้วิธีใช้แฮนด์จับ
ขั้นตอนต่อไป: ค้นหาเวโลซีแรปเตอร์ที่ให้ความร่วมมือ

📏 การวัดจักรวาลด้วยไม้บรรทัด IKEA

เมื่อติดไม้บรรทัด IKEA ต่อกัน ในทางทฤษฎีจะเป็นไปได้ที่จะวัดเส้นรอบวงของโลก
ขีดจำกัดปัจจุบัน: เราขาดไม้บรรทัดไป 40,075,000 เมตร

🎲 สถิติที่ไม่เป็นวิทยาศาสตร์

82% ของผู้ตอบแบบสอบถามชอบเปอร์เซ็นต์ที่คิดค้นขึ้นเอง
63% ของผู้คนอ่านข้อความนี้แล้วพยักหน้า
100% ของประโยคก่อนหน้านี้ถูกสร้างขึ้นโดยไม่มีข้อมูลใดๆ
